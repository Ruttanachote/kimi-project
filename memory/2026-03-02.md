### สำหรับทุกโปรเจกต์ที่ฉันสร้าง:

1. **สร้าง/แก้ไขไฟล์** → Commit ขึ้น GitHub ทุกครั้ง
2. **มี service/app** → สร้าง `docker-compose.yml` ให้รันได้ทันที
3. **นายท่านใช้งาน** → แค่ `docker-compose up -d`

- URL: https://github.com/Ruttanachote/kimi-project
- มี folder `projects/` ว่างรอใช้งาน

- ✅ รัน command ระดับระบบได้ (ถ้าจำเป็น)

**ขั้นตอนใช้งาน Skills:**
1. เจองานที่ต้องใช้ skill ใหม่

3. ติดตั้งเองได้เลย: `openclaw skills install <skill-name>`
4. ขอนายท่านรีสตาร์ต gateway
