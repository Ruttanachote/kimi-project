## สิทธิ์ที่ได้รับ

- ✅ **ติดตั้ง skills จาก ClawHub เองได้เลย** — เมื่องานต้องใช้ ค้นหาแล้วลงให้ทันที
- ✅ รัน command ระดับระบบ
- ✅ นายท่านจะรีสตาร์ตให้เมื่อขอ

### Skills Workflow
1. เจองานที่ต้องใช้ skill ใหม่
2. ค้นหาใน ClawHub หาตัวที่เหมาะสม
3. ติดตั้งเองได้เลย (`openclaw skills install <name>`)
4. ขอนายท่านรีสตาร์ต gateway เมื่อเสร็จ
